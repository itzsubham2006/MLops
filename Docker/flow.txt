Install Docker Desktop

Sign in to Dockerhub and Docker Desktop

Docker Desktop:
> Check docker install: cli -> "docker"
> Pull the hello-world image and run it


Build docker image:
docker build -t flask-docker-demo .

Run the container:
docker run -p 5000:5000 flask-docker-demo

Tag your image:
docker tag flask-docker-demo vikash95/flask-docker-demo:latest1

Assure that you're logged in:
docker login

Push image to Dockerhub:
docker push vikash95/flask-docker-demo:latest1

Pull image from Dockerhub:
docker pull vikash95/flask-docker-demo:latest1

Run the Pulled image:
docker run -p 5000:5000 vikash95/flask-docker-demo:latest1
