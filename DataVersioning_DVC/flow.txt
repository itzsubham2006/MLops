1. Create a git repo and clone it in local.

2. Create code.py and add code to it. (it will save a csv file to a new "data" folder. ).

3. Do a git add, commit, push before initializing the dvc.

4. "pip install dvc" and do "dvc init", it will create files (.dvcignore, .dvc).

5. Now do "mkdir s3" (creates a new directory).

6. Now we do "dvc remote add -d myremote s3"

7. Next "dvc add data/"
    Now do ("git rm -r --cached data" and "git commit -m "Stop tracking data"")
    because initially we were tracking data from git now we are tracking from dvc.

8. Again we do "dvc add data/"

9. Now do "dvc commit" and then "dvc push"

10. Now again do git add-commit-push to mark this stage as first version of data.

11. Now write a new code to code.py to add more rows and column to the data and check using "dvc status"

12. Now do "dvc commit-push" to add new version of data and it will give new id corresponds to new data. And changes will be saved to s3.
